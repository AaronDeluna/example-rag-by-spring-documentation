# Spring AI API

## Введение

Spring AI API охватывает широкий спектр функциональных возможностей. Каждая основная функция подробно описана в своем собственном разделе. Для предоставления общего обзора доступны следующие ключевые функции:

### AI Model API

Портативный `Model API` для различных поставщиков AI для моделей `Chat`, `Text to Image`, `Audio Transcription`, `Text to Speech` и `Embedding`. Поддерживаются как `синхронные`, так и `потоковые` варианты API. Также поддерживается доступ к специфическим функциям модели.

![Иерархия моделей, ширина=900, выравнивание="центр"](model-hierarchy.jpg)

С поддержкой AI моделей от OpenAI, Microsoft, Amazon, Google, Amazon Bedrock, Hugging Face и других.

![выравнивание="центр", ширина="800px"](spring-ai-chat-completions-clients.jpg)

### Vector Store API

Портативный `Vector Store API` для нескольких поставщиков, включая новый `SQL-подобный API фильтрации метаданных`, который также является портативным. Доступно 14 векторных баз данных.

### Tool Calling API

Spring AI упрощает вызов ваших сервисов моделями AI в качестве методов, аннотированных `@Tool`, или объектов POJO `java.util.Function`.

![Основная последовательность действий для вызова инструментов, ширина=500, выравнивание="центр"](tools/tool-calling-01.jpg)

Проверьте документацию по xref::api/tools.adoc[Tool Calling].

### Автоконфигурация

Автоконфигурация Spring Boot и стартеры для AI моделей и векторных хранилищ.

### ETL Data Engineering

ETL фреймворк для Data Engineering. Это обеспечивает основу для загрузки данных в векторную базу данных, помогая реализовать паттерн Retrieval Augmented Generation, который позволяет вам передавать ваши данные модели AI для их включения в ответ.

![выравнивание="центр"](etl-pipeline.jpg)
