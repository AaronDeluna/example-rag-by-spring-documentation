# API модели речи

[ПРИМЕЧАНИЕ]
====
Эта страница была заменена новой документацией по синтезу речи (TTS).

Пожалуйста, обратитесь к xref:api/audio/speech.adoc[API синтеза речи (TTS)] для получения актуальных общих интерфейсов (`TextToSpeechModel` и `StreamingTextToSpeechModel`).

Старые классы, специфичные для провайдеров (`SpeechModel`, `StreamingSpeechModel`, `SpeechPrompt`, `SpeechResponse`), были удалены в пользу общих интерфейсов, которые работают со всеми провайдерами TTS (OpenAI, ElevenLabs и будущими провайдерами).
====
