## Пути к конфигурационным файлам IDE

Полное описание содержится по [ссылке](https://www.jetbrains.com/help/idea/directories-used-by-the-ide-to-store-settings-caches-plugins-and-logs.html).

Пути для Linux (с поправкой на версию 2025.2):
- **исполняемые файлы приложения IDE**: `~/.local/ide/idea-IC-252.25557.131/bin/`
- **Configuration directory**: `~/.config/JetBrains/.config/JetBrains/IdeaIC2025.2/`
- **System directory**: `~/.cache/JetBrains/IdeaIC2025.2/`
- **Logs directory**: `cd ~/.cache/JetBrains/IdeaIC2025.2/log/`
- **Plugins directory**: `~/.local/share/JetBrains/IdeaIC2025.2/`
